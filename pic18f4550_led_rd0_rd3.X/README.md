# PIC18F4550 - LED Sequence

**Descripción:** Secuencia básica de LEDs en RD0-RD3 y RB0. Demostración mínima de control de puertos GPIO con animación de LEDs.

## Características
- Secuencia de LEDs: RD0 → RD1 → RD2 → RD3 → RB0 → RB1
- Delay de 5ms entre cada paso
- Fosc = 4MHz (cristal HS)

## Periféricos Utilizados
- GPIO (LEDs)
