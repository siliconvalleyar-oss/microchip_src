# pic18f4550_ws2812_adafruit

Proyecto para microcontrolador PIC18F4550

## Microcontrolador

- **PIC18F4550**

## Estructura del proyecto

```
pic18f4550_ws2812_adafruit.X/
├── src/           # Codigo fuente
├── inc/           # Archivos de cabecera
├── examples/      # Ejemplos de uso
├── nbproject/     # Configuracion MPLAB X
├── Makefile
└── README.md
```

## Dependencias

- MPLAB X IDE
- Compilador XC8 (PIC18F) / XC32 (PIC32MX)

## Compilacion y Programacion

Abrir en MPLAB X IDE y seleccionar "Make and Program Device".
