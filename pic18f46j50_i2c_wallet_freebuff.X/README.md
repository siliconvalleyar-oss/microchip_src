# pic18f46j50_i2c_wallet_freebuff

Proyecto para microcontrolador PIC18F46

## Microcontrolador

- **PIC18F46**

## Estructura del proyecto

```
pic18f46j50_i2c_wallet_freebuff.X/
├── src/           # Codigo fuente
├── inc/           # Archivos de cabecera
├── examples/      # Ejemplos de uso
├── nbproject/     # Configuracion MPLAB X
├── Makefile
└── README.md
```

## Dependencias

- MPLAB X IDE
- Compilador XC8 (PIC18F) / XC32 (PIC32MX)

## Compilacion y Programacion

Abrir en MPLAB X IDE y seleccionar "Make and Program Device".
