# PIC32MX795F512H - E-Paper Display v1.3

**Descripción:** Tercera versión del driver para pantallas E-Paper Pervasive Displays. Configuración refinada con soporte para display 2.66" (296x152, frame buffer 5624 bytes).

## Características
- Soporte para pantallas EPD (1.54" a 4.37")
- SPI 8MHz, sistema 80MHz
- Configuración de pines vía estructura pins_t
- Registros de inicialización para display 2.66"

## Periféricos Utilizados
- SPI4, GPIO
