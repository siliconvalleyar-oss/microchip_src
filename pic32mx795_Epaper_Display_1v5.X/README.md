# PIC32MX795F512H - E-Paper Display v1.5 (Barcode)

**Descripción:** Quinta versión del driver E-Paper con soporte para generación de códigos de barras Code 39. Incluye fuente Terminal16x24e y configuración completa para displays Pervasive Displays de 1.54" a 4.37".

## Características
- Generación de código de barras Code 39 (clase Barcode con drawCode39)
- Fuente Terminal 16×24 (glyphs bitmap 16×24)
- Soporte para pantallas: 1.54", 2.13", 2.66", 2.71", 2.87", 3.70", 4.17", 4.37"
- SPI 8MHz, sistema 80MHz
- Basado en Pervasive Displays Library Suite

## Periféricos Utilizados
- SPI4, GPIO
