# PIC32MX795F512H - MRF24J40 Wireless Transceiver

**Descripción:** Driver y aplicación para comunicación inalámbrica IEEE 802.15.4 usando módulo MRF24J40 con PIC32MX795F512H. Implementación para redes de sensores inalámbricos (WSN).

## Características
- Comunicación inalámbrica IEEE 802.15.4 a 2.4GHz
- Configuración de PAN ID, dirección, canal
- Modos TX y RX
- SPI para comunicación con MRF24J40
- Sistema 80MHz

## Periféricos Utilizados
- SPI, GPIO, MRF24J40
